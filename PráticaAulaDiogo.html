Ferramentas Utilizadas

O VS é uma ferramenta do tipo editor de texto. Ele possui funcionalidades para melhorar o desempenho e qualidade do código em HTML, como por exemplo o Live Server que permite que você possa ver suas páginas enquanto as desenvolve, ou Emmet que auxilia na produção do código com algumas pré-definições. O Inspetor de elementos é uma ferramenta que existem dentro dos navegadores. Através dele você pode inspecionar, para finalidade de entender como funciona o site ou para estudos, dentro você também pode ver como foi feito às margens, quais scrips estão rodando ou estilos de css.

Estrutura básica de HTML

O HTML funciona como um texto que é regido por tags(HTML, Body, p, title...), as tags são abertas por "<" e fechadas por ">", isto para definir umas tag dentro do HTML, o conteúdo fica entre duas tags uma que fecha (quando ela possui uma barra na frente) e outra que abre a formatação. As tags possuem funções específicas, em resumo para iniciar, pode iniciar pela tag HTML (dá início a página e as normas que irá seguir), seguido head (pré-carrega antes de aparecer para o usuário) e body (a pagina que vai aparecer para o usuário é colocada aqui), esse é a estrutura básica de HTML.

Tags

As tags são elementos para formatar o texto do HTML, colocar em forma de título, mudar o texto para itálico, pular linha, possúi diversas tags. É possível adicionar a elas atributos como por exemplo links, mudar a cor da letra, imagens entre outros.

Listas

Existem dois tipos de listas presentes, as ordenadas que são atribuídas a tag ol e as não ordenada que se utiliza a tag ul, dentro das tags do tipo de lista, é utilizado a tag li que será o conteúdo da lista em questão, importante informar que é possível incluir uma lista dentro de outra como por exemplo os menus de alguns sites. Veja a seguir exemplos de listas:
Lista Ordenada

  1. Primeiro elemento da lista
  2. Segundo elmento da lista
  3. Terceiro elemento da lista
  4. Quarto elemento da lista
Agora lista desordenada:
  • neste caso não
  • tem uma sequência
  • de números na
  • frente da lista

Links

Os links são tags direcionado para navegação, como por exemplo ir para próxima página ou navegar dentro da própria página, podem ser utilizados também para ir até outros sites, de forma que será direcionado dentro da própria aba ou abrir uma aba nova para ação.
